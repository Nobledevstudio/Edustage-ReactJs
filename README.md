# Edustage - Online Education Platform

![Edustage Logo](src/assets/logo.png)

**Edustage** is a modern, responsive online education platform built using **React** and **Vite**. It provides students with access to courses, features like testimonials, registration, and more, all wrapped in a clean, professional UI with smooth animations.

## Features

- Fully **responsive design** optimized for desktop, tablet, and mobile.
- Animated **hero section** with scroll-triggered animations using **Framer Motion**.
- Course registration with **countdown timer**.
- Interactive **testimonial cards**.
- Smooth scrolling navigation.
- Professional UI components: Hero, Features, Courses, About, Contact, Footer.
- Mobile-friendly **hamburger menu**.
- Contact form integration.
- Clean and maintainable **React component structure**.

- **Frontend:** React, Vite, CSS, Framer Motion

## Installation

1..  **Clone the repository:**

git clone https://github.com/Nobledevstudio/Edustage.git
cd Edustage

2.. **Install dependencies:**
  npm install
3.. **Run the development server:**
  npm run dev

4. Open http://localhost:5173 to view the project.

## Usage

1. Use the navigation bar to scroll to different sections of the page.

2. Click “Learn More” or “See Courses” in the hero section to explore the platform.

3. Fill out the registration form to simulate course registration.

4. Interact with testimonials and feature cards to see animations.
